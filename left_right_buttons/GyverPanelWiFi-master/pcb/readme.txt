Вариант печатной платы, разработанной пользователем RailWar (Evgeny)
для варианта устройства на ESP8366 (плата ESP-12F) по схеме "С полным набором компонент (отключение матрицы - MOSFET)" 
с toush-button смотри статью "Варианты устройств. Схемы." в wiki.

Для данной платы используется
#define DEVICE_VARIANT 1
в настройках прошивки
